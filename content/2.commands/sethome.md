# Sethome

A simple homes plugin with the ability to create, delete, and teleport to many different homes.

## Commands

3 Type to trust player

```bash
/sethome name
/delhome name
```

## Note

- Max 5 for rank [Member]<br />
- Max 10 for rank [Co-Chief]
