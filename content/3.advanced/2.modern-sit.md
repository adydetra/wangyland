# Modern Sit

Ultimate Sit, Chair and Lay.

## Features

- Users can sit on a stairs.
- Users can sit on a slab.

## Commands

```bash
/sit #sit on a Block
/lay #lay on a Block
/bellyflop #bellyFlop on a Block
/spin #spin on a Block
/crawl #crawl on the ground
```
